# SEPH.PRINCIPAL Frontend

Aplicación administrativa Angular 21 standalone, preparada para ejecutarse sin backend mediante mocks locales.

## Ejecutar

```bash
npm install
npm start
```

URL:

```text
http://localhost:4200
```

Credenciales de demo:

```text
admin@seph.local
Admin123456!
```
